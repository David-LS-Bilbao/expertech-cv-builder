# EXPERTECH CV

## Descripción corta

Aplicación web orientada a la creación y gestión de currículums con un enfoque práctico, claro y pensado para un MVP de aprendizaje.

## Estado

Bootcamp JavaScript MVP

## Objetivo del proyecto

Desarrollar una primera versión funcional de una herramienta que permita estructurar, editar y presentar un CV digital de forma sencilla, con una base sólida para futuras mejoras en experiencia de usuario, personalización y exportación.

## Stack previsto

- HTML5
- CSS3
- JavaScript
- Arquitectura modular en frontend
- Git y GitHub para control de versiones

## Roadmap resumido

- Definir la estructura base del proyecto
- Implementar la maquetación inicial de la interfaz
- Desarrollar módulos JavaScript para la gestión de datos del CV
- Conectar la lógica de formulario y la vista previa
- Preparar una primera versión MVP lista para demo
